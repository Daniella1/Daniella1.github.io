# Pre-Deployment Checklist

Before deploying your website, make sure to complete these tasks:

## Essential Updates

- [x] Update `_config.yml` with your actual information:
  - [x] title (your name)
  - [x] email
  - [x] description
  - [x] github_username
  - [x] linkedin_username
  - [x] youtube_username

- [ ] Update all page content:
  - [x] index.md (home page)
  - [x] about.md (bio and background)
  - [x] cv.md (education, experience, skills)
  - [?] publications.md (add your papers)
  - [-] projects.md (showcase your projects)
  - [-] teaching.md (add teaching materials)
  - [x] contact.md (your contact info)

- [ ] Add your profile photo to `assets/images/profile.jpg`

- [ ] Set up Formspree contact form:
  - [ ] Create account at formspree.io
  - [ ] Update form action URL in contact.md

## Content to Add

- [ ] Write at least 1-2 blog posts
- [ ] Add project images to `assets/images/projects/`
- [ ] Upload CV PDF to `assets/cv/`
- [ ] Create project pages in `_projects/` folder
- [ ] Add publication details

## Optional Enhancements

- [ ] Set up Google Analytics (if desired)
- [ ] Add newsletter subscription (Mailchimp, etc.)
- [ ] Create custom 404 page
- [ ] Add social media meta tags
- [ ] Set up RSS feed customization
- [ ] Add comments system (Disqus, Giscus, etc.)

## Testing

- [ ] Test locally with `bundle exec jekyll serve`
- [ ] Check all links work
- [ ] Verify images load correctly
- [ ] Test contact form
- [ ] Check mobile responsiveness
- [ ] Validate HTML/CSS

## Deployment

- [ ] Create GitHub repository named `yourusername.github.io`
- [ ] Push code to GitHub
- [ ] Enable GitHub Pages in repository settings
- [ ] Wait for site to deploy (2-3 minutes)
- [ ] Visit your site at https://yourusername.github.io
- [ ] Set up custom domain (if purchased)

## After Launch

- [ ] Share on LinkedIn
- [ ] Add to email signature
- [ ] Share with colleagues/friends
- [ ] Submit to Google Search Console
- [ ] Plan regular content updates

---

Remember: You can launch with minimal content and add more over time. Don't wait for perfection!
